Magic Carpet Battle Arena (Unity version)
-----------------------------------------
A Unity-based game where two players ride magic carpets and fight against each other in a small arena.



Running the game:
- Execute UnityCarpet.exe



Controls:
- Up - Accelerate
- Down - Decelerate
- Left - Turn left
- Right - Turn right
- A - Punch left
- D - Punch right
